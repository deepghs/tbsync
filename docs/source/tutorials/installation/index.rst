Installation
===================

tbsync is currently hosted on PyPI. It required python >= 3.7.

You can simply install tbsync from PyPI with the following command:

.. code:: shell

    pip install tbsync

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/deepghs/tbsync@main
